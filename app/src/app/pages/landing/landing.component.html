Develop UI
herramientas para el desarrollador

Construye rápidamente componentes SINTRA
sin tener que buscar en otro sitio.

Guía para el desarrollo de aplicaciones:

Develop UI es un conjunto de herramientas de diseño que ayudan al desarrollador a seguir una línea única de contrucción de interfaces.

Components

Todos los componentes en un sólo sitio, preparados para ser implantados sin necesidad de buscar.
Angular Material y PrimeNG suman para formar un solo estilo.

Blocks

Bloques de interfaz de usuario preconstruidos.
Ejemplos que ya forman parte en varias aplicaciones y que repiten fórmulas de maquetación.

Styles

Tokens de diseño utilizados para cada elemento, contrucción y variables.

Wiki

Documentación para normas diseño
